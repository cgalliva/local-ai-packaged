# Coping Styles - Word Clusters

## Structure Guide

```yaml
Hierarchy:
  Style > Subcategory > Aspect > Keywords
  
Aspects (when data available):
  IN: Internal experience ("I feel...")
  OUT: External/relational ("They/it...")
  ANT: Antidote/resolution (opposite state)
  
  Note: Some categories use single "Keywords" section when IN/OUT 
        distinction was not provided in source materials

Keywords:
  Core: More likely to indicate specific style
  Shared: Overlaps with other categories (cross-refs in parentheses)

Notation:
  [PARENT] = subcategory of parent style
  → = resolves to
```
